# CHOOSEEASY

Project for Git & GitHub Hands-On Assignment.

## Description
This project demonstrates version control using Git and GitHub. It includes a server-client architecture.

## Commands Used
- `git init`
- `git add`
- `git commit`
- `git branch`
- `git merge`
- `git remote`
- `git push`

## Features
- Client-side React application
- NodeJS Server
- REST API integration
